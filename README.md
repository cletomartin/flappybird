# flappybird

Clon de Flappy Bird escrito en Python.

## Instrucciones

1. Generar el virtual environment (si no lo has hecho nunca):

   ```
   python3 -m venv ./venv
   ```

1. Activar el virtual environment:

   ```
   . ./venv/bin/activate
   ```

1. Ejecutar videojuego:

   ```
   python flappybird/flappy_bird.py
   ```
